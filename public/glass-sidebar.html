Welcome!

This is a simple layout with a glass sidebar and a clean content area. Read the CSS comments to learn how each piece works.

How the glass works

The sidebar has a semi-transparent background plus backdrop-filter: blur(). The gradient page background shows through, creating a frosted glass look.

Try edits

Change the blur amount, background opacity, or the gradient colors to see the effect change instantly.

Responsive

Resize the window. On small screens, the layout stacks so it stays easy to read.
